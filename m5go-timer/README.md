
# ⏱ M5GO Countdown Timer

M5GOデバイスを使用して制作したカウントダウンタイマーです。  
ボタン入力による時間設定とカウントダウン機能を実装しました。

---

## 🎥 Demo Video

https://www.youtube.com/shorts/1vdjIuWwS6U

---

## 📌 プロジェクト概要

このタイマーは以下の用途を想定して設計しました。

- 目覚まし時計
- 料理タイマー
- 学習タイマー

時間が0になるとアラームが鳴り、  
スヌーズ機能を使用すると再度カウントダウンが開始されます。

---

## ⚙️ 機能

- 時・分・秒の設定
- カウントダウンタイマー
- アラーム通知
- スヌーズ機能
- RGBライト通知

---

## 🧠 実装内容

- ボタン入力による時間設定
- カウントダウンロジック設計
- ループ処理による時間更新
- スヌーズ機能の実装
- アラーム音とRGBライト制御

---

## 🛠 使用技術

- M5GO
- Embedded Programming
