## 🌐 Webプロジェクト

### フラワーショップECサイト開発

PHPとMySQLを用いて、花のオンラインショップを開発しました。  
商品一覧、詳細表示、購入機能などの基本的なEC機能を実装しています。

また、ユーザーが直感的に操作できるよう、  
**画面構成や導線設計を意識したUI設計**にも取り組みました。

- 商品一覧 → 詳細 → 購入までの流れをシンプルに設計
- 視認性・操作性を意識したレイアウト構成
