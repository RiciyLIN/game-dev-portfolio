## 🎮 プロジェクト紹介

### VRアクションゲーム「アロード (Arrowd)」

弓矢を使って道路を生成し、  
馬車をゴールまで導くVRアクションゲームです。  

大学のチーム開発プロジェクトとして制作しました。

---

### 🧠 担当（Responsibilities）

本プロジェクトでは、主にゲームロジックの実装を担当しました。

- 馬車の移動制御（ルートに沿った移動・速度調整・挙動制御）
- ワープ処理の実装（プレイヤーの向きや移動方向の制御を含む）
- 道路生成ロジック（矢による動的なオブジェクト生成とスケール制御）
- オブジェクト間の当たり判定と状態管理

---

### ⚙ 技術的な工夫・課題解決

開発中、オブジェクトの挙動が不安定になる問題に直面しました。  
その際、処理を細分化し、条件ごとにログを確認しながら原因を特定しました。

特に、

- 移動処理と回転処理の分離  
- 状態遷移の明確化  

を行うことで、安定した動作を実現しました。  

これにより、ゲーム全体の挙動の安定性を向上させることができました。
