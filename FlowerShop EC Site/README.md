## 🌐 Webプロジェクト

# フラワーショップECサイト開発

PHPとMySQLを用いて、花店をテーマとしたECサイトを個人開発しました。  
商品一覧表示、商品詳細確認、カート追加、購入履歴管理など、ECサイトとしての基本機能を実装しています。

また、セッション管理を利用したログイン機能や、数量変更に応じた価格計算処理を実装し、  
実際の利用を想定したデータ管理や画面遷移を意識して設計しました。

ユーザーが直感的に操作できることを重視し、  
UI/UX設計にも取り組みました。

---

## 🛠 使用技術

- PHP
- MySQL
- HTML / CSS
- JavaScript
- XAMPP

---

## ✨ 実装機能

- 商品一覧表示機能
- 商品詳細モーダル表示
- カート追加機能
- 購入履歴管理
- ログイン / ログアウト機能
- セッション管理
- 数量変更時の価格計算

---

## 💡 工夫した点

- 商品一覧 → 商品詳細 → カート追加 → 購入までの導線をシンプルに設計し、  
  ユーザーが迷わず操作できるUIを意識しました。

- モーダル表示を採用することで、  
  ページ遷移を減らし操作性向上を図りました。

- 視認性を意識し、  
  レイアウトや配色を調整しました。

- セッション管理を利用し、  
  ログイン状態やカート情報を保持できるよう実装しました。
