## 🌐 Webプロジェクト

### フラワーショップECサイト開発

PHPとMySQLを用いて、花店をテーマとしたECサイトを個人で開発しました。  
商品一覧表示、カート機能、購入履歴の管理など、基本的なEC機能を実装しています。

また、セッション管理を用いたログイン機能や、数量に応じた価格計算など、  
実際の利用を想定した設計を意識しました。

さらに、ユーザーが直感的に操作できるよう、  
画面構成や導線設計を意識したUI/UX設計にも取り組みました。

- 商品一覧 → 詳細 → カート追加 → 購入の流れをシンプルに設計  
- モーダル表示による商品詳細確認で操作性を向上  
- 視認性を意識したレイアウトと配色設計
